# Sugaryfish 的弹幕姬 1.1.0

## 更新亮点

- 重构整体视觉排版与双主题细节，统一 Light / Dark 的字重、行高、分割线和交互反馈。
- 新增自动获取 `SESSDATA` 按钮，可通过应用内登录窗口直接获取并保存登录态。
- 新增沉浸式挂机 HUD：鼠标离开后自动淡出顶部控制区与页脚，仅保留核心弹幕内容和状态呼吸灯。
- 修复沉浸模式顶部留白问题，隐藏 HUD 时会真实释放空间，弹幕流自动上吸。
- 礼物消息改为单行极致压缩布局，降低高频礼物对普通弹幕阅读节奏的挤压。
- 新增同用户同礼物连击堆叠逻辑，连续送礼会直接累加 `xN` 数字并触发微弹动效。
- 新增大额礼物 / 醒目留言置顶告示牌，支持顶部 12 秒稳定驻留与进度条退场。
- 优化弹幕列表自动贴底、鼠标悬停锁定和滚动位置感知，减少阅读历史时的视口抖动。
- 优化直播间人气轮询显示，维持更稳定的实时刷新体验。

## 适合写在 GitHub Release 的简版

1. 新增应用内自动获取 `SESSDATA`。
2. 完整重构 Light / Dark 双主题与沉浸挂机模式。
3. 礼物消息支持单行压缩、连击堆叠和大额置顶告示牌。
4. 优化弹幕滚动、顶部 HUD、窗口细节与整体阅读节奏。
